# Mô hình mạng LAN – Cisco Packet Tracer

## Mô tả
Xây dựng mạng LAN cho doanh nghiệp nhỏ với 2 VLAN, 
cấu hình Trunk, Router-on-a-Stick và DHCP tự động.

## Công nghệ sử dụng

- VLAN, Trunk
- Router-on-a-Stick
- DHCP
